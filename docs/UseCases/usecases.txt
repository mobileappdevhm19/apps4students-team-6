Use Cases

1. Man kann ein Event erstellen:
Beliebige Event kann durch einen Nutzer erstellt werden.

2. Man kann andere Events in der App suchen und finden:
Events sind kategoriersiert und für jeden ist etwas dabei.

3. Man kann ein eigenes Profil mit Interessen erstellen:
vorlieben und Interessen können im Profil hinzugefügt werden, um bessere vorschläge zu bekommen.

4. Man kann sich beliebige Profiles ansehen:
Zugang zu anderen profile um Interessen andere mitglieder anzusehen.

5. Man kann sich anmelden:
Ein Login ist für die nutzung der app erforderlich.

6. Man kann an einem von jemanden bereits erstellten Event teilnehmen:
Durch klicken auf "Sign in" meldet sich man für einen bestimmten Event.
